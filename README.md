# CodeLingua

Aplicación web educativa para aprender **programación e inglés técnico** al mismo tiempo.

## Objetivo
Fomentar el aprendizaje dual combinando código real y vocabulario técnico en inglés.

## Tecnologías
- HTML5  
- CSS3  
- JavaScript  
- JSON local  
- Publicado en GitHub Pages
